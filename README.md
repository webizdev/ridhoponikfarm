# Ridhoponic Farm - Web E-Commerce & Admin Dashboard

Modern organic minimalist e-commerce platform for Ridhoponic Farm, specialized in premium hydroponic vegetables from Sumedang.

## Features
- **Responsive Design**: Mobile-first approach for both customer site and admin dashboard.
- **Dynamic Content**: Hero, About, and Contact sections editable via Admin.
- **Product Catalog**: Categorized items (Harvest vs Supplies) with search/filter.
- **Cart System**: Seamless shopping experience.
- **WhatsApp Integration**: Automated order placement via WhatsApp.
- **Admin Dashboard**: Full CRUD for products, content management, and order tracking.
- **Demo Mode**: Currently runs on Local Storage for portablity, but ready for MySQL integration.

## Tech Stack
- HTML5 / CSS3 (Vanilla)
- JavaScript (Vanilla)
- PHP (API Layer - optional for production)
- MySQL (Database - optional for production)
- Docker (Containerization)

## How to Run
### Demo Mode
Simply open `index.html` in any modern browser. Data is saved in your browser's local storage.

### Production Mode (Docker)
1. Ensure Docker is installed.
2. Run `docker-compose up -d`.
3. Access the site at `http://localhost:8080`.
4. Access phpMyAdmin at `http://localhost:8081`.

## Admin Login
- **URL**: `admin.html`
